History

HISTORY OF IDE:

Eclipse was starting to be created in the late 1990s.
The goal of Eclipse was to bring developers closer to Java-based middleware.
Was based on the idea of having a open and free platform.
In computer programming, Eclipse is an integrated development environment (IDE).
It contains a base workspace and an extensible plug-in system for customizing the environment.
Written mostly in Java, Eclipse can be used to develop applications.

Keyboard Shortcuts




Ctrl + F11 ----------- Save and launch Application

F11 ------------------ Debug

F4 ------------------- Show tooltip

Ctrl + Shift + / ----- add comment

Ctrl+= --------------- zoom in

Ctrl+- -------------- zoom out

Alt +/ --------------- word completer

Ctrl + Alt + Up ------ Duplicate Line

Ctrl + Shift + L------ Shortcut list
